# Интеллектуальный Гладиатор (VKR Project) 

Приложение для проведения научных дебатов с ИИ-агентами, использующее RAG (Retrieval-Augmented Generation) для предоставления аргументации, основанной на трудах великих мыслителей.

## О проекте 📖

"Интеллектуальный Гладиатор" — это интерактивная платформа, где пользователь может сразиться в красноречии и логике с цифровыми воплощениями великих философов (Кант, Сократ, Фрейд и др.). Проект демонстрирует интеграцию современных ИИ-технологий, векторных баз данных и мультимедийных возможностей.

### Основные возможности ✨
- **Реалистичные дебаты**: Оппоненты используют специфический стиль и аргументацию своих исторических прототипов.
- **RAG-поддержка**: ИИ обращается к локальной базе знаний (ChromaDB) или выполняет веб-поиск (DuckDuckGo) для подкрепления своих слов реальными цитатами и фактами.
- **Голосовое управление**: Распознавание речи (SpeechRecognition) позволяет вести дискуссию голосом.
- **Озвучка ответов**: Тексты озвучиваются с помощью Edge-TTS и gTTS с поддержкой различных эмоций и тембров.
- **Генерация отчетов**: После каждого боя формируется подробный PDF-отчет со стенограммой и вердиктом жюри.
- **Система прогресса**: Личный кабинет с игровой экономикой (аргуконы), статистикой побед и магазином новых оппонентов.

## Технологический стек 🛠️

- **Язык**: Python 3.10+
- **GUI**: PySide6 (Qt for Python).
- **LLM**: Интеграция с DeepSeek API.
- **Векторная БД**: ChromaDB (локальное хранилище эмбеддингов).
- **NLP**: sentence-transformers (для поиска релевантности).
- **TTS/STT**: Edge-TTS, speech_recognition, pygame mixer.
- **Data**: SQLite (хранение профилей и статистики).
- **Report**: ReportLab (генерация PDF).

## Структура проекта 📂

- `src/` — Исходный код приложения.
- `ui/` — Файлы дизайна Qt Designer (.ui).
- `data/` — Локальное хранилище БД SQLite.
- `bin/` — Необходимые бинарные файлы (ffmpeg).
- `assets/` — Графические ресурсы и звуки.
- `audio_cache/` — Кэш сгенерированной озвучки.

## Установка и запуск 🚀

1. **Клонируйте репозиторий**:
   ```bash
   git clone https://github.com/username/vkr-project.git
   cd vkr-project
   ```

2. **Настройте виртуальное окружение**:
   ```bash
   python -m venv venv
   source venv/bin/activate  # Для Windows: venv\Scripts\activate
   pip install -r requirements.txt
   ```

3. **Настройте окружение (.env)**:
   Создайте файл `.env` в корне проекта и добавьте ваш API ключ:
   ```env
   DEEPSEEK_API_KEY=your_key_here
   ```

4. **Запустите приложение**:
   ```bash
   python src/app_controller.py
   ```
